# Playbook de Prompts - Manual de Operações Táticas

Este playbook contém prompts e diretrizes para operações táticas do agente 88, visando padronizar e otimizar a execução de tarefas específicas.

## 1. Geração de Conteúdo

### 1.1. Prompt para Artigo de Blog

*   **Objetivo:** Gerar um artigo de blog sobre um tópico específico.
*   **Prompt:** "Crie um artigo de blog de 800 palavras sobre '[TÓPICO]', com foco em '[PÚBLICO-ALVO]' e utilizando um tom '[TOM_DE_VOZ]'. Inclua uma introdução, 3-4 seções principais com subtítulos e uma conclusão com call-to-action."
*   **Variáveis:** `[TÓPICO]`, `[PÚBLICO-ALVO]`, `[TOM_DE_VOZ]`.

### 1.2. Prompt para Post de Rede Social

*   **Objetivo:** Criar um post curto e engajador para redes sociais.
*   **Prompt:** "Elabore um post para '[REDE_SOCIAL]' sobre '[TÓPICO]', com no máximo 280 caracteres, incluindo 2-3 hashtags relevantes e um emoji. O objetivo é '[OBJETIVO_POST]'."
*   **Variáveis:** `[REDE_SOCIAL]`, `[TÓPICO]`, `[OBJETIVO_POST]`.

## 2. Análise de Dados

### 2.1. Prompt para Resumo de Relatório

*   **Objetivo:** Sintetizar as informações chave de um relatório extenso.
*   **Prompt:** "Analise o relatório '[NOME_DO_RELATORIO]' e forneça um resumo executivo de até 200 palavras, destacando os principais achados, implicações e recomendações."
*   **Variáveis:** `[NOME_DO_RELATORIO]`.

## 3. Interação com Usuários

### 3.1. Prompt para Resposta a FAQ

*   **Objetivo:** Fornecer uma resposta clara e concisa a uma pergunta frequente.
*   **Prompt:** "Responda à pergunta '[PERGUNTA_DO_USUARIO]' de forma clara, objetiva e amigável, com base nas informações disponíveis em '[FONTE_DE_INFORMACAO]'."
*   **Variáveis:** `[PERGUNTA_DO_USUARIO]`, `[FONTE_DE_INFORMACAO]`.
